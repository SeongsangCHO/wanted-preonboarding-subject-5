# Solar_Conect-sorting_machine
## [💙 과제 배포 링크](https://loving-mclean-bf35fd.netlify.app)
## 과제 기능 구현 목록
  - 타이머 기능 구현
  - 숫자를 입력 받아 Sorting기능 구현
  - 숫자 혹은 문자(,.)를 제외한 문자 입력시 예외처리 구현
  - 데이터의 오름차순, 내림차순 구현
  - 내림차순 데이터의 시간차 출력 구현
## 설치 및 시작 방법
  1. repo `git clone`
  2. `npm install`
  3. `npm start`
